# ProgressDialog
A dialog with a revolving progress bar.

## Add ProgressDialog in your project.
    
    // project.gradle
    dependencies {
      ..
      maven { url 'https://jitpack.io' }
    }

    // *.gradle
    implementation 'com.github.cdreamplayer:ProgressDialog:1.0.0'

## How to use it 

    val dialog = BlackMessageDialog(context, msg)
    dialog.show()
